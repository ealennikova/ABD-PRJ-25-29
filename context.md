В книге приводится обоснование уровней стресса, описаны изменения и признаки для каждого уровня.
Для ускорения анализа книги проведён парсинг совпадений по исследуемым признакам и их синонимам, все совпадения выгружены с контекстом 600 символов в файлы .csv с соответствующим названием для дальнейшей работы над проектом.
